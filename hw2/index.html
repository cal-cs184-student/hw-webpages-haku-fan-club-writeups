Part 4: Edge flip

To implement the edge flip operation, we first visualized the mesh before and after a flip, labeling every half-edge, vertex, edge, and face, as shown in the diagram below. Then, we reassigned the halfedge pointer for every vertex, edge, and face. After this, we used the Halfedge::setNeighbors method to reassign the correct next, twin, vertex, edge, and face pointers for every half-edge. As recommended, we setting the pointers of all the elements in the mesh even if they did not change for clarity and ease when implementing and debugging. Initially, it was difficult understanding which pointers to reassign, but after clearly drawing out all elements before and after modification, we were able to implement the edge flip operation pretty smoothly.
